Cheat Sheet — Glosario de HTML

Este glosario presenta las etiquetas y atributos principales de HTML con definiciones y ejemplos, basado en el material de clase.

<!DOCTYPE html>

Define el tipo de documento y le indica al navegador que el archivo está escrito en HTML5.

<!DOCTYPE html>

Nota: siempre debe ir en la primera línea del documento.

<html>

Elemento raíz de un documento HTML. Todo el contenido se encierra dentro de esta etiqueta.

<html lang="es"> ... </html>

<head>

Contiene metadatos del documento: título, codificación, enlaces a CSS, scripts, etc.

<head> <title>Mi página</title> </head>

<body>

Contiene todo el contenido visible de la página web, como texto, imágenes, enlaces y formularios.

<body> <h1>Hola mundo</h1> </body>

Encabezados (<h1>...<h6>)

Definen títulos y subtítulos, con importancia jerárquica (h1 el más importante, h6 el menor).

<h1>Título principal</h1> <h2>Subtítulo</h2>

<p>

Representa un párrafo de texto.

<p>Este es un párrafo.</p>

<a> (hipervínculos)

Crea un enlace a otra página, recurso o sección interna.

<a href="https://ejemplo.com">Visitar</a>

Una URL puede ser absoluta (con protocolo y dominio) o relativa (en relación a la ubicación del archivo actual).

Listas (<ul>, <ol>, <li>, <dl>, <dt>, <dd>)

<ul> crea listas no ordenadas. <ol> crea listas ordenadas. <li> define cada elemento. <dl> listas descriptivas con términos <dt> y descripciones <dd>.

<ul><li>Elemento</li></ul>

Tablas (<table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot>)

Permiten organizar datos en filas y columnas.

<table><tr><th>Nombre</th><td>Carlos</td></tr></table>

Multimedia (<img>, <audio>, <video>, <source>)

Insertan imágenes, audio o video en una página.

<img src="foto.jpg" alt="Descripción">

Objetos (<iframe>, <object>)

<iframe> permite incrustar páginas externas. <object> inserta otros tipos de archivo (ej. PDF).

<iframe src="https://ejemplo.com" width="400" height="300"></iframe>

Formularios (<form>, <input>, <select>, <option>, <textarea>, <label>)

Permiten capturar datos del usuario mediante controles.

<form><label>Nombre: <input type="text" name="nombre"></label></form>

El atributo <code>name</code> establece el nombre de la variable y <code>value</code> su valor.

Atributos comunes

  • id: Nombre único dentro del documento.
  • class: Permite agrupar elementos para estilos o scripts.
  • name: Define el nombre de una variable en formularios.
  • value: Establece un valor en campos de formularios.
